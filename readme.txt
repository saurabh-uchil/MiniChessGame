It is a mini chess game with only three type of pieces i.e knights,rooks and bishops.
Both the players has to decide the maximum number of moves before the game starts.
The number of moves is then decided as the average of the number of moves selected by both the players.
The players who captures all the pieces of the opponent or the player with the maximum points by the end of the last move wins the game. 
This game is designed using Java fx and with respect to the OOP concepts.
